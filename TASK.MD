# Завдання для новачка

- [ ] **Ознайомлення з NPM**
  - Дізнатися, що таке npm.
  - Розібратися, як встановлюються бібліотеки та плагіни.
  - Ініціалізувати `package.json` у проекті.

- [ ] **Dev Server**
  - Встановити який небудь дев спервер для локального запуску сайту. Щоб не в вебштормі відкриваловася сайт, а на якому небудь іншому встановленому в package.json(наприклад, `serve`, `live-server` або `vite`).
  - Налаштувати команду запуску (наприклад, `npm start`) у `package.json`.

- [ ] **Swiper**
  - Замінити існуючий самописний слайдер на бібліотеку [Swiper](https://swiperjs.com/).
  - Переконатися, що слайдер працює коректно.

- [ ] **Day.js та форматування дат**
  - Встановити бібіліотеку [day.js](https://day.js.org/).
  - На сторінці епізодів змінити формат дати з "December 16, 2013" на "16 december 2013", використовуючи метод `format`.
